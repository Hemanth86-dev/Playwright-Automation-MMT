# Playwright MakeMyTrip Automation Framework

## Project Overview

This project is built using Playwright and JavaScript for automating web application testing.

It includes:

- UI Automation Testing
- Page Object Model (POM)
- Reusable Utilities
- Screenshots on Failure
- HTML Reports
- GitHub Actions CI/CD
- Git + GitHub Version Control

## Tech Stack

- Playwright
- JavaScript
- Node.js
- Git
- GitHub
- GitHub Actions

## Project Goal

To automate real-world workflows like:

- Login
- Hotel Search
- Booking Flow
- Payment Validation
